# TEXT RPG ENGINE (Data-Driven)

**Text RPG Engine** — це консольний ігровий рушій, написаний мовою **Python 3.13**. Проєкт демонструє архітектурний підхід **Data-Driven**, де вся ігрова логіка відокремлена від контенту. Рівні, вороги та предмети завантажуються із зовнішніх **JSON-файлів**.

---

## ОСОБЛИВОСТІ

- **Модульна архітектура**: чіткий поділ на `Engine`, `Player`, `Loader` та `Data`.
- **JSON-конфігурація**: додавання нових кімнат та монстрів без зміни коду.
- **Бойова система**: покрокові битви з елементальними типами *(Вогонь > Темрява > Вода)*.
- **Інвентар**: система слотів *(Main Hand / Off Hand)* та витратних предметів.
- **Збереження**: повна серіалізація стану героя у `savegame.json`.

---

## ЯК ЗАПУСТИТИ

Вам знадобиться встановлений **Python 3.10** або новіше.

1. Клонуйте репозиторій або скачайте архів:
   ```bash
   git clone https://github.com/TimofeyTofanchuk/text-rpg-engine.git
   ```

2. Перейдіть у папку проєкту:
   ```bash
   cd text-rpg-engine
   ```

3. Запустіть гру:
   ```bash
   python main.py
   ```

---

## УПРАВЛІННЯ

Гра керується текстовими командами *(регістр не має значення)*:

- `go forward` / `go back` — переміщення по світу.
- `take` — підібрати предмет у кімнаті.
- `inventory` або `i` — відкрити рюкзак.
- `equip [назва]` — взяти зброю в руки або випити зілля.
- `attack` — атакувати ворога *(тільки в бою)*.
- `save` — зберегти прогрес.

---

## РОЗШИРЕННЯ ГРИ

Щоб додати нову кімнату, відкрийте файл `data/world.json` і додайте новий блок:

```json
{
    "stage_id": 5,
    "name": "Нова Кімната",
    "description": "Опис...",
    "loot": ["iron_sword"],
    "enemy_id": "goblin_bomber"
}
```
